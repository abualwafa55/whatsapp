# تطبيق WhatsApp Desktop - دليل التشغيل

## المتطلبات الأساسية

### 1. تثبيت Python
- Python 3.8 أو أحدث
- تحميل من: https://www.python.org/downloads/

### 2. تثبيت Node.js
- Node.js 14 أو أحدث
- تحميل من: https://nodejs.org/

## خطوات التثبيت والتشغيل

### طريقة 1: التشغيل التلقائي (موصى بها)

1. **تشغيل ملف الإعداد (مرة واحدة فقط):**
   ```bash
   .\setup.bat
   ```

2. **تشغيل التطبيق:**
   ```bash
   .\run.bat
   ```
   هذا الملف سيشغل السيرفر والتطبيق معاً تلقائياً

### طريقة 2: التشغيل المنفصل (للتحكم الأفضل)

1. **تشغيل Baileys Server (في نافذة منفصلة):**
   ```bash
   .\start-server.bat
   ```
   ⚠️ اترك هذه النافذة مفتوحة

2. **تشغيل التطبيق Python (في نافذة جديدة):**
   ```bash
   .\start-app.bat
   ```

### طريقة 3: التشغيل اليدوي

#### الخطوة 1: تثبيت مكتبات Python
```bash
pip install -r requirements.txt
```

#### الخطوة 2: إعداد Baileys Server
```bash
cd baileys-server
npm install
```

#### الخطوة 3: تشغيل Baileys Server
```bash
cd baileys-server
node server.js
```

#### الخطوة 4: تشغيل تطبيق Python
في نافذة terminal جديدة:
```bash
python whatsapp_app.py
```

## كيفية الاستخدام

1. **إنشاء جلسة جديدة:**
   - أدخل اسم للجلسة (مثل: `my-session`)
   - اضغط على "إنشاء جلسة"

2. **مسح QR Code:**
   - افتح WhatsApp على هاتفك
   - اذهب إلى الإعدادات > الأجهزة المرتبطة
   - امسح QR Code المعروض في التطبيق

3. **إرسال رسالة:**
   - أدخل رقم الهاتف (مثل: 962791234567)
   - اكتب الرسالة
   - اضغط "إرسال"

## حل المشاكل الشائعة

### المشكلة: لا يظهر QR Code
- تأكد من أن Baileys Server يعمل على المنفذ 3000
- تحقق من سجل الأحداث في التطبيق

### المشكلة: فشل الاتصال بـ API
- تأكد من تشغيل Baileys Server أولاً
- تحقق من أن المنفذ 3000 غير مستخدم

### المشكلة: خطأ في إرسال الرسالة
- تأكد من أن الرقم صحيح مع رمز الدولة
- تحقق من أن الجلسة متصلة (أخضر)

## ملاحظات مهمة

- يتم حفظ بيانات الجلسات في مجلد `baileys-server/sessions`
- لا تشارك ملفات الجلسة مع أحد
- يمكن ربط عدة أجهزة بنفس الحساب

## الدعم الفني

للمشاكل والاستفسارات، تحقق من:
- سجل الأحداث في التطبيق
- ملف `baileys-server/logs`
